Solution 1: The Uber Reademe

If you're lucky....

Install Party

  • Install Webserver (Apache,Ngnix,IIS)
  • Install CFML Engine (Adobe, Railo, Bluedragon)
  • Install DB (MSSQL, MySQL, Oracle, etc)
  • Install caching software (Couchbase, MongoDB, Terracotta)

Config Party (oh wait, theres more)

  • Configure Webserver
  • Configure CFML Engine
  • Configure DB Server
  • Configure cache

Setup Party (nope, not done yet)

  • Get Code (clone repo, FTP, etc)
  • Restore DB, run scripts
  • Configure all the software to point to code

PROBLEM

  • Prone to human error

  • Quickly drop out of sync production
  • Requires special maintance
  • Heavy Maintance
  • Time Consuming
  • Platform specfic
